# How to use the app


## 1. Install dependencies

```bash
    pip install -U yt-dlp 
    yt-dlp --version

    # Follow the second steps if you prefer pytube.
    pip install pytube 
    # or 
    pip3 install pytube 

    # If you don't have pip, you can install it using 
    pip install pip
    # or upgrade using 
    pip install --upgrade pip
```

## 2. Downloading

- Run this code to download videos. 

```bash
    python3 ytalt.py "insert youtube url here"

    # alternative (using pytube)
    python3 downloadyt.py "insert youtube url here"

```
